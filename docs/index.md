# Projeto Databricks — Arquitetura Medalhão

## 📌 Sobre o Projeto

Este projeto foi desenvolvido utilizando o Databricks Free Edition com arquitetura Medalhão.

O objetivo é construir um pipeline de dados completo utilizando:

- Landing
- Bronze
- Silver
- Gold

com automação através de Jobs & Pipelines.

---

## 🏗️ Arquitetura Utilizada

```text
Landing -> Bronze -> Silver -> Gold
```

---

## ⚙️ Tecnologias Utilizadas

- Databricks
- Apache Spark
- Delta Lake
- Python (PySpark)
- GitHub
- MkDocs

---

## 🚀 Fluxo do Pipeline

```text
Banco de Dados
      ↓
Landing
      ↓
Bronze
      ↓
Silver
      ↓
Gold
```
