# 🥈 Camada Silver

## 📌 Objetivo

A camada Silver é responsável pelo tratamento e qualidade dos dados dentro da arquitetura Medalhão.

Nesta etapa os dados deixam de ser apenas dados brutos e passam a se tornar informações mais confiáveis.

---

## ⚙️ Processos Realizados

Durante o processamento da camada Silver foram aplicadas regras de Data Quality.

Entre os principais tratamentos estão:

- remoção de dados inválidos
- padronização de formatos
- tratamento de valores nulos
- validação de informações
- organização dos dados

---

## 🧹 Data Quality

O objetivo principal da camada Silver é garantir qualidade e consistência.

As validações ajudam a evitar:

- inconsistências
- duplicidades
- erros de formatação
- dados incompletos

---

## 🚀 Benefícios

A utilização da camada Silver proporciona:

- maior confiabilidade
- melhor organização
- facilidade de análise
- preparação para modelagem analítica

---

## 📊 Resultado

Após o processamento da camada Silver os dados ficam preparados para alimentar a camada Gold, onde são utilizados para análise e modelagem dimensional.
